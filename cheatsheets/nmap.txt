All -s_ option can be concatenated in a single -s (example -sCVUT)

-sC Script Scan
-sV Version Enumeration
-pX,Y-Z -p- Select ports (X), port range (Y-Z) or all ports (-p-). If unspecified, 1000 most common ports are scanned
-sU UDP scan
-sT TCP scan (used by default)
-sUT -p U:X,Y-Z,T:X,Y-Z Scan specified UDP and TCP ports
